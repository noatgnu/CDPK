N-Glycosylation Sequon Parser

N-Glycosylation Sequon Parser is a tool that can be used to search and identify sequence containing the N-glycosylation sequon in MS peptide summary file. If the summary also includes HexNAc modification, it would try to identify sequence with HexNAc modification.

CPS is a tool that can be used to compare 2 protein summary outputs from mass spectrometry and identifies the differences in proteomic content of the two.
